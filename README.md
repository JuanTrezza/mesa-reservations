## Sobre el proyecto

**MESA** es una aplicación web de reservas de restaurante con una interfaz premium y elegante. Permite a los clientes seleccionar fecha, hora, número de comensales y sección del restaurante, brindando una experiencia de reserva sofisticada acorde al ambiente del establecimiento.

## Tecnologías utilizadas

| Tecnología | Descripción |
|---|---|
| **React 19** | Biblioteca UI con hooks modernos |
| **TypeScript** | Tipado estático para mayor robustez |
| **Tailwind CSS 3** | Estilos utilitarios con paleta personalizada |
| **Vite 6** | Build tool ultrarrápido |
| **Lucide React** | Iconografía consistente |
| **Framer Motion** | Animaciones fluidas |
| **Google Fonts** | Playfair Display · Cormorant Garamond · Inter |

## Características

- Selección de fecha con calendario interactivo
- Elección de franja horaria (almuerzo / cena)
- Configuración de número de comensales
- Selección de sección del restaurante (salón principal, terraza, barra, privado)
- Resumen de reserva en tiempo real
- Diseño oscuro premium con acentos dorados
- Totalmente responsive

## Ejecución local

**Requisitos:** Node.js 18+

```bash
npm install
npm run dev
```

La app estará disponible en `http://localhost:3000`.

## Build y despliegue

```bash
npm run build   # genera ./dist
npm run preview # previsualiza el build
```

El despliegue a GitHub Pages se realiza automáticamente al hacer push a `main` mediante GitHub Actions.
